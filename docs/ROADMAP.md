# Stitch Manager - Roadmap

## Objetivo

Construir un sistema de gestión de producción para una bordadora publicitaria mientras se aprende desarrollo Full Stack de forma profesional.

---

# Fase 0 - Fundamentos

Objetivo:
Entender las herramientas básicas de desarrollo.

Aprender:

- Terminal
- Git
- GitHub
- Node.js
- npm/pnpm
- package.json
- Arquitectura básica de proyectos

Entregables:

- Proyecto inicial creado
- Repositorio Git
- Estructura de carpetas

---

# Fase 1 - APIs y Async/Await

Objetivo:
Aprender cómo una aplicación obtiene y modifica datos.

Aprender:

- JSON
- HTTP
- REST
- GET
- POST
- PUT
- DELETE
- fetch
- async
- await

Herramientas:

- json-server

Entregables:

- API local
- CRUD de clientes
- CRUD de órdenes
- CRUD de producción

---

- API REST funcional
