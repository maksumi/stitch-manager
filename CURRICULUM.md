CURRICULUM

Propósito

Este documento define el temario de referencia para el aprendizaje técnico dentro del proyecto.

No representa un roadmap obligatorio ni un orden fijo de estudio.

Las necesidades del proyecto determinarán cuándo estudiar cada tema.

El objetivo es disponer de una equivalencia entre el bootcamp original basado en Java/Spring y el ecosistema JavaScript/TypeScript utilizado en Stitch Manager.
